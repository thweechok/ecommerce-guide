1

นำเข้าข้อมูลคำสั่งซื้อ

▼

ที่แถบเมนูด้านซ้าย ในหมวด E-Commerce คลิกเมนู คำสั่งซื้อ แล้วคลิกปุ่ม นำเข้าคำสั่งซื้อ สีเขียว ที่มุมขวาบน

  1. คลิกปุ่ม นำเข้าคำสั่งซื้อ สีเขียว ที่มุมขวาบน
  2. หน้าต่าง อัปโหลดข้อมูลคำสั่งซื้อ จะเด้งขึ้นมา
  3. เลือกวิธีนำเข้าไฟล์ (เลือกวิธีใดวิธีหนึ่ง):
    • แท็บ ไฟล์ — ลากไฟล์ Excel มาวางในกรอบเส้นประ หรือคลิกเลือกจากเครื่อง (.xls .xlsx .csv) ★ แนะนำ
    • แท็บ โฟลเดอร์ Drive — วาง Drive Folder URL ในช่อง → เลือกเดือน/ปี → กดปุ่ม ค้นหาไฟล์
    • แท็บ Excel URL — วาง URL ไฟล์ Google Sheets ลงในช่อง
  4. เลือกไฟล์ที่ต้องการนำเข้า (ถ้าใช้แท็บ "ไฟล์" ให้คลิก Dropdown เลือก Platform: TikTok / Shopee / Lazada)
  5. กดปุ่ม อัปโหลด X ไฟล์ สีเขียว
  6. รอระบบประมวลผล — จะมีแถบ progress bar แสดงความคืบหน้า
  7. เมื่อเสร็จจะแสดง อัปโหลดสำเร็จ ✅ พร้อมสรุปผล → กดปุ่ม เสร็จสิ้น

  3. เลือกวิธีนำเข้าไฟล์ (เลือกวิธีใดวิธีหนึ่ง):
    • แท็บ ไฟล์ — ลากไฟล์ Excel มาวาง หรือคลิกเลือกจากเครื่อง ★ แนะนำ
    • แท็บ โฟลเดอร์ Drive — วาง Drive Folder URL → เลือกเดือน/ปี → กดปุ่ม ค้นหาไฟล์
    • แท็บ Excel URL — วาง URL ไฟล์ Google Sheets ลงในช่อง
  4. เลือกไฟล์ที่ต้องการนำเข้า แล้วกดปุ่ม อัปโหลด X ไฟล์ สีเขียว
  5. รอระบบประมวลผล — จะมีแถบ progress bar แสดงความคืบหน้า
  6. เมื่อเสร็จจะแสดง อัปโหลดสำเร็จ ✅ พร้อมสรุปผล → กดปุ่ม เสร็จสิ้น

นำเข้าข้อมูลเงินเข้าธนาคาร

▼

ที่แถบเมนูด้านซ้าย ในเมนู E-Commerce คลิกเมนู รับเงินเข้าธนาคาร แล้วกดปุ่ม รับเงินเข้าธนาคารทั้งหมด สีเขียว แล้วเลือก นำเข้ารับเงินจากไฟล์

  1. ที่แถบเมนูด้านซ้าย ในเมนู E-Commerce คลิกเมนู รับเงินเข้าธนาคาร
  2. กดปุ่ม รับเงินเข้าธนาคารทั้งหมด (สีเขียว) แล้วเลือก นำเข้ารับเงินจากไฟล์
  3. หน้าต่าง นำเข้ารายการถอนเงิน จะเด้งขึ้นมา เลือก Platform: TikTok หรือ Shopee
    • เลือก ไฟล์ Excel / PDF — ลากไฟล์มาวาง หรือคลิกเลือกไฟล์ ★ แนะนำ
    • เลือก Google Drive — วาง URL ของ Google Drive Folder
  4. กดปุ่ม นำเข้าข้อมูล
  5. ตรวจสอบข้อมูล → กด บันทึก
  6. เมื่อเสร็จจะแสดง เสร็จสิ้น ✅

นำเข้าบิลค่าใช้จ่ายแพลตฟอร์ม

▼

ที่แถบเมนูด้านซ้าย ในเมนู E-Commerce คลิกเมนู ค่าใช้จ่ายแพลตฟอร์ม แล้วกดปุ่ม นำเข้าบิลค่าใช้จ่าย สีเขียว

  1. ที่แถบเมนูด้านซ้าย ในเมนู E-Commerce คลิกเมนู ค่าใช้จ่ายแพลตฟอร์ม
  2. กดปุ่ม นำเข้าบิลค่าใช้จ่าย (สีเขียว)
  3. หน้าต่าง AI Scan บิล จะเด้งขึ้นมา เลือกวิธีนำเข้า (เลือกวิธีใดวิธีหนึ่ง):
    • แท็บ โฟลเดอร์ Drive — วาง URL โฟลเดอร์ที่มีไฟล์ PDF → เลือกแพลตฟอร์ม → กดปุ่ม สแกน + แม็ปบัญชีอัตโนมัติ ★ แนะนำ
    • แท็บ ไฟล์เดี่ยว — ลาก PDF / รูปมาวาง แล้วกด สแกน หรือ รับข้อมูลบัญชี
  4. รอระบบ AI ประมวลผล
  5. เมื่อเสร็จจะแสดง เสร็จสิ้น ✅
